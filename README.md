# AI-WAF Diploma

Система защиты веб-приложений с использованием искусственного интеллекта: разработка модели обнаружения и предотвращения атак.

## Цель проекта

Разработать прототип web application firewall (WAF), который:
- анализирует HTTP-запросы,
- обнаруживает атаки с помощью методов машинного обучения,
- комбинирует rule-based фильтры и ML-классификатор,
- может быть встроен между reverse-proxy и веб-приложением.
